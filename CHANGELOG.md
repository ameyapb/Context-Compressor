# Changelog

## 0.1.8

- Added Team Tracker: a per-member board for tracking tasks, notes, and links across a sprint, persisted in VS Code global state.
- Added SQLite Viewer: browse SQLite and .db files in an interactive table panel with search, column filters, sorting, and pagination, all running locally without any external tools.
- Added Line Filter: filter any open file by regex pattern, chain multiple filters, and view per-step match counts in the sidebar history.
- Added Prompt Templates: save reusable instruction blocks with a {{FILES}} placeholder, edit them in the VS Code editor, and open them in a fillable preview panel.
- Added context compression modes: strip comments, collapse whitespace, or extract signatures only, with a live token-savings percentage shown per file.
